# Architecture

## Components

1. **DevContainer**: Provides the runtime environment.
2. **GitHub Actions**: Automates the "Template -> New Repo" transformation.
3. **Templates (`templates/`)**: Stores the genetic code for new projects (AGENTS + AI scaffold memory).
4. **Docs (`docs/`)**: Reference notes and decisions.
5. **AGENTS.md**: Entry point for AI instructions and links to context sources.
